1.3. HTML-Literatur

1.1 Erfolgsprinzipien des WWW 1.2 HTTP 1.3. HTML-Literatur 1.4. HTML Wireframe

1) Was ist HTML?

HTML steht für Hyper Text Markup Language und ist eine Standard-Auszeichnungssprache für Webseiten. HTML beschreibt die Struktur einer Webseite und besteht aus einer Reihe von Elementen. HTML-Elemente teilen dem Browser mit, wie der Inhalt angezeigt werden soll. D.h. wir können damit eigene Website erstelle.

2) Wie kann man eine geschachtelte geordnete Liste der Schachtelungstiefe 3 erzeugen?

Die Elemente der geordneten Liste sollen Listen enthalten und die Elemente dieser Listen sollen auch wieder Listen enthalten.

3) Wie ist eine HTML-Tabelle aufgebaut?

Eine Tabelle besteht aus Tabellenzellen innerhalb von Zeilen und Spalten und sie wird duch <table> definiert. Jede Tabellenzeile beginnt mit einem <tr> und endet mit einem </tr> und jede Tabellenzelle wird durch a <td> und </td> definiert. Alles zwischen <td>und </td>ist der Inhalt der Tabellenzelle. Tabellenüberschriften erfolgt duch <th> und </th>.

4) Welche Konventionen sollte man bei Pfaden und Dateinamen beachten?

Bei Dateinamen sollte man deutsche Umlaute und scharfes S vermeiden. Als Sonderzeichen ist der Unterstrich "_" erlaubt.
Andere Satzzeichen sollte man vermeiden. Auch auf Leerzeichen soll man unbedingt verzichten. Die Pfaden sollte man in Anführungszeichen schreiben.

5) Wie baut man in HTML ein Menü?

Durch die HTML-Listen-Tags:
<ul> definiert eine ungeordnete Liste.
<ol> definiert eine geordnete Liste.

6) Welche Attribute sollte man bei Bildern wie verwenden?

Das src-Attribut: gibt den Pfad (URL) zum Bild an, und das alt-Attribut: stellt einen alternativen Text für das Bild.
