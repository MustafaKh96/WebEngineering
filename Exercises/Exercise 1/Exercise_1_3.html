1.3. HTML-Literatur

1) Was ist HTML?

HTML steht für Hyper Text Markup Language und ist eine Standard-Auszeichnungssprache für Webseiten. HTML beschreibt die Struktur einer Webseite und besteht aus einer Reihe von Elementen. HTML-Elemente teilen dem Browser mit, wie der Inhalt angezeigt werden soll. D.h. wir können damit eigene Website erstelle.

2) Wie kann man eine geschachtelte geordnete Liste der Schachtelungstiefe 3 erzeugen?

Indem die Elemente (Items) einer geordneten Liste wieder eine Liste enthalten und diese auch wieder als Elemente eine Liste benutzt.

4) Welche Konventionen sollte man bei Pfaden und Dateinamen beachten?

Man sollte in jedem Fall deutsche Umlaute und scharfes S in den Dateinamen vermeiden. Als Sonderzeichen ist in jedem Fall der Unterstrich "_" erlaubt. Andere Satzzeichen sollten Sie vermeiden. Fragezeichen "?", Sternzeichen "*", Istgleichzeichen "=" darf man auf keinen Fall in Dateinamen verwenden. Auch auf Leerzeichen soll man unbedingt verzichten.
