1.2 HTTP

1.1 Erfolgsprinzipien des WWW 1.2 HTTP 1.3. HTML-Literatur 1.4. HTML Wireframe

1) Sie bekommen im Browser den HTTP Status Code 200. Was bedeutet das?

Dies bedeutet, dass die Anfrage erfolgreich bearbeitet wurde.

2) Sie bekommen im Browser den HTTP Status Code 301. Was hat das zu bedeuten?

Das bedeutet, dass die vom Nutzer angeforderten Daten nicht mehr in der besuchten Internetadresse zu finden sind, sondern dauerhaft verschoben wurden.

3) Sie bekommen im Browser den HTTP Status Code 400. Was hat das zu bedeuten? Was können Sie dagegen tun?

Dies bedeutet, dass die Anfrage aufgrund eines Fehlers des Benutzers (z. B. die Anfrage ist syntaktisch falsch) nicht bearbeitet werden kann.
Um das Problem zu lösen, könnte man die Webseite erneut laden, oder einen anderen Browser verwenden.

4) Sie bekommen im Browser den HTTP Status Code 403. Was hat das zu bedeuten?

Dies bedeutet, dass die Anfrage aufgrund einer fehlenden Berechtigung vom Benutzer nicht durchgeführt werden kann.
