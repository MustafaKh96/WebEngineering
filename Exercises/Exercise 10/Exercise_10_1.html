Gibt es in TypeScript statische Klassen? Erläutern Sie den Unterschied von TypeScript zu anderen objekt-orientierten, statisch typisierten Programmiersprachen wie Java und C#.

Nein, es gibt in TypeScript keine statische Klassen.
TypeScript ist nicht wie Java oder C#, daher sollte man nicht erwarten, dass die gleichen Konzepte von Java oder C# in TypeScript auftauchen.
Mit C# können keine Funktionen außerhalb von Klassen definiert werden. In TypeScript ist dies jedoch möglich.
