6.5 DeepCopy

6.1 Klammerpaare 6.2 Topologische Iterierbarkeit 6.3 Topologischer Generator 6.4 Proxy 6.5 DeepCopy

Schreiben Sie eine rekursive Funktion deepCopy( struct ) als ES6-Ausdruck, so dass beliebig geschachtelte Arrays und Objekte struct tiefenkopiert werden können.
Verwenden Sie zu diesem Zweck den konditionalen ternären Operator, Array.map(), Object.fromEntries() und Object.entries().
Verwenden Sie dabei nur Arrow Functions und Ausdrücke, keine Anweisungen, keine Blöcke.
Verwenden Sie nicht die JSON-Methoden. 

 
        const deepCopy = (struct) =>
        {
            var erg;
            if(Array.isArray( struct )){
                erg = [];
                erg = struct.map(x => x);
                return erg;
            }
            else if(!Array.isArray(struct) && typeof struct === "object"){
                erg = {};
                for (let pos in struct) {
                    let wert = struct[pos];
                    erg[pos] = deepCopy(wert);
                }
                return erg;
            }
            return struct;
        }

        const myObject = {vorname: "David", nachname: "Peter", myarray: [[10, 20, 30, 40],[[1, 2]],[[0, 1],[0, 1]]]};
        console.log(myarray)
        console.log(deepCopy(myarray));
